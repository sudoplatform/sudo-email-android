//sudoemail/com.sudoplatform.sudoemail/SudoEmailClient/listEmailAddresses

listEmailAddresses

Get a ListOutput of EmailAddresses. If no EmailAddresses can be found, the ListOutput will contain null for the ListOutput.nextToken field and contain an empty list.

Return

A list of EmailAddresses or an empty list if no email addresses can be found.

Parameters

cachePolicy
Determines how the data will be fetched. When using CachePolicy.CACHE_ONLY, be aware that this will only return cached results of similar exact API calls.
filter
Filter the email addresses so that only those that match all of the values of the fields in the filter are returned.
limit
Number of EmailAddresses to return. If omitted the limit defaults to 10.
nextToken
A token generated from previous calls to listEmailAddresses. This is to allow for pagination. This value should be generated from a previous pagination call, otherwise it will throw an exception. The same arguments should be supplied to this method if using a previously generated nextToken.
sudoId
The identifier of the Sudo that owns the EmailAddress.
abstract suspend fun listEmailAddresses(sudoId: String?, limit: Int, nextToken: String?, cachePolicy: CachePolicy, filter: () -> EmailAddressFilter??): ListOutput<EmailAddress>
